# 스마트 물류창고 운영 데이터 기반 출고 지연 시간 예측 모델 개발

## 1. 대회 주제
스마트 물류창고 운영 데이터를 기반으로 **향후 30분간의 평균 출고 지연 시간(분)** 을 예측하는 모델을 개발한다.

---

## 2. 배경
AMR(자율이동로봇) 기반 스마트 물류창고가 빠르게 확산되고 있다.  
하지만 로봇 대수, 주문량, 배터리 상태, 통로 혼잡도 등 다양한 운영 요인이 복합적으로 작용하여, 출고 지연을 사전에 예측하고 대응하는 것은 여전히 어려운 과제이다.

---

## 3. 데이터셋 개요
- 총 2,000개의 독립적인 창고 운영 시나리오를 시뮬레이션하여 생성
- 각 시나리오는 약 6시간
- 총 25개 타임슬롯(15분 간격)으로 구성
- 주문 유입, 로봇 운영, 배터리/충전, 혼잡도, 패킹 병목 등 다양한 요인이 복합적으로 작용
- 비선형적인 지연 패턴을 보이는 데이터셋

---

## 4. 데이터 파일 구성

train.csv              # 학습 데이터 (250,000행 × 94컬럼)  
test.csv               # 평가 데이터 (50,000행 × 93컬럼)  
layout_info.csv        # 창고 레이아웃 보조 정보 (300행 × 15컬럼)  
sample_submission.csv  # 제출 양식 파일 (50,000행 × 2컬럼)  

---

## 5. 데이터 구조 상세 설명

### 5.1 train.csv
모델 학습을 위한 데이터셋으로, 각 행은 특정 시나리오의 특정 시점에서의 창고 운영 상태를 나타낸다.

- ID : 샘플 고유 식별 번호
- layout_id : 창고 레이아웃 ID
- scenario_id : 시나리오 ID
- 90개 피처 컬럼 : 해당 시점의 창고 운영 상태
- avg_delay_minutes_next_30m : 향후 30분간의 평균 출고 지연 시간(분) → 예측 대상

---

### 5.2 test.csv
모델 성능 평가를 위한 데이터셋으로, train.csv와 동일한 구조이나 타깃 컬럼이 제외되어 있다.

- ID : 평가 데이터 샘플 고유 식별 번호
- layout_id : 창고 레이아웃 ID
- scenario_id : 시나리오 ID
- 90개 피처 컬럼 : train.csv와 동일

---

### 5.3 layout_info.csv
창고 레이아웃 관련 보조 테이블이다.

- layout_id : 창고 레이아웃 ID
- 14개 레이아웃 관련 컬럼

---

### 5.4 sample_submission.csv
최종 결과 제출을 위한 샘플 제출 파일이다.

- ID : 평가 데이터(test) 고유 식별 번호
- avg_delay_minutes_next_30m : 예측값(분)

---

## 6. 과제
주어진 창고 운영 스냅샷 데이터(로봇 상태, 주문량, 배터리, 혼잡도 등 90개 피처)를 분석하여  
각 시점에서 향후 30분간 출고되는 주문들의 평균 지연 시간을 예측하는 모델을 개발한다.

- 목표: 물류 운영 최적화를 위한 지연 예측 모델 개발

## 10. 평가 방식 (리더보드)

### 10.1 평가 지표: MAE (Mean Absolute Error)

```python
import numpy as np

def MAE(true, pred):
    score = np.mean(np.abs(true - pred))
    return score
```
